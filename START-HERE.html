Dadda Multiplier Design

This repository contains two files:

  • design.sv: This file contains the Verilog code for the Dadda multiplier design.
  • testbench.sv: This file contains the Verilog code for the Dadda multiplier testbench.

The explanations for the design and testbench files are provided in separate files:

  • design-explain.txt: This file contains an explanation of the design file.
  • testbench-explain.txt: This file contains an explanation of the testbench file.

There is also a log file (testbench.log) that shows the terminal output of running the testbench.
